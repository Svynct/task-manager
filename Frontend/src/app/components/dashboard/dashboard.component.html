Bem-vindo!

Este sistema emula um Gerenciador de Tarefas. Nele você pode organizar tarefas, logar com seu usuário (administrador ou normal), onde o Administrador pode cadastrar, editar e deletar outros usuários.

Este sistema foi idealizado respeitando os princípios RESTful e foi desenvolvido em .NET Core e Angular 12. Nele você poderá realizar o CRUD completo numa Web API.
